分析手法・透明性レポート

🔍 AIスコアリングの仕組みを完全公開 📋 バックテスト期間・前提条件を明示
ℹ︎ このページの目的
  • HG Analyticsのスコア算出方法・重み付けを完全公開します
  • 分析モデルの前提・限界・バックテスト条件を透明化します
  • 本サイトの情報は参考情報であり、投資助言・投資推奨ではありません
  • 金融商品取引法第2条に基づき、断定的判断の提供は行っていません

🎯 総合スコアの構成

各銘柄の総合スコアは「ファンダメンタル」「テクニカル」「マクロ環境」の3要素で構成されます。
中長期バリュー選定を重視した設計で、企業の本質的価値（ROE・FCF・利益率）を優先評価します。

📈 ファンダメンタル分析（最大22点）65%
📊 テクニカル分析 — エントリー補助（最大12点）35%
🌎 マクロ環境調整（±5点）加算

ファンダ65% / テクニカル35%。テクニカルは主に「いつ買うか」のエントリータイミング補助として機能します。

📈 ファンダメンタル分析の採点基準（最大22点）

指標最大配点評価基準重視理由
ROE（自己資本利益率）+3ptROE≥20%:+3 / ≥15%:+2 / ≥8%:+1企業の稼ぐ力。バフェットが最重視する指標
FCFマージン（フリーCF）+3ptFCF率≥20%:+3 / ≥12%:+2 / ≥5%:+1 / 赤字:-2キャッシュ創出力。会計操作不可能な実力値
売上成長率+3pt+20%以上:+3 / +10%:+2 / +3%:+1成長の持続性。長期リターンの源泉
利益率（営業利益率）+3pt≥20%:+3 / ≥12%:+2 / ≥5%:+1競争優位性（ワイドモート）の証拠
PER（株価収益率）+2ptPER 5-40倍圏で割安ほど高得点バリュエーション確認
PBR（株価純資産倍率）+2ptPBR<1.0で+2（簿価割れ）、<1.5で+2資産価値対比の割安度
配当利回り+2pt≥4%:+2 / ≥2%:+1株主還元・財務健全性のシグナル
PEG比率+2ptPEG<0.8:+2 / <1.5:+1 / >4.0:-1成長を加味したバリュエーション
財務健全性（負債比率）+2pt負債比率<30%:+2 / <80%:+1 / >300%:-1不況耐性・借入コスト負担
流動比率+1pt>2.0:+1 / <1.0:-1短期支払い能力

※ データはyfinance経由で取得。日本株ではFCF・PEG等が取得できない場合があります（その場合は0点として扱います）。

📊 テクニカル分析の採点基準（最大12点・エントリー補助）

テクニカルは「何を買うか」ではなく「いつ買うか」の補助指標として位置づけています。 売られすぎサイン（RSI低水準・VWAP割れ）を特に重視します。

指標最大配点評価基準
RSI（相対力指数）+2ptRSI<30:+2（売られすぎ） / 30-45:+2 / 45-60:+1 / 75-85:-1 / ≥85:-3（旧-5から緩和）
MACD+2ptヒストグラムプラス:+2 / マイナスはペナルティなし（押し目の可能性）
VWAP乖離+2ptVWAP割れが深いほど高得点（割安エントリーのサイン）
出来高×VWAP複合+2pt出来高急増×VWAP割れで機関仕込みシグナル
トレンド+1pt上昇トレンド:+1 / 強い下降:-1（旧-2から緩和）
ボリンジャーバンド+1pt下限付近:+1（上限超えはペナルティなし）
1週間騰落+1pt+2%超:+1 / マイナスはペナルティなし（押し目の可能性）
1ヶ月騰落+1pt+10%超:+1（中期モメンタム確認）

※ テクニカル指標は直近3ヶ月の日足データで算出（yfinance）。リアルタイム値ではありません。

🌎 マクロ環境調整（±5点）

米10年債利回り・VIX・USD/JPY を総合判断し、リスクオン/オフを判定します。

環境調整値判定条件
リスクオン最大+3点VIX<15、低金利環境
中立0点VIX 15-25、金利安定
リスクオフ最大-4点VIX>25、急激な金利上昇

加えて、円安局面の日本輸出株への加点、高金利×高PER銘柄への追加減点も行います。

📋 バックテスト・実績評価の条件

項目条件
評価期間選定日から5営業日後の終値で判定
勝利条件5日後騰落率 ≥ +0.1%
対象銘柄日本株20銘柄 + 米国株10銘柄のウォッチリスト
データソースYahoo Finance API（yfinance）
スプレッド・手数料未考慮（実際の取引コストが発生します）
ベンチマーク比較日本株：日経225 / 米国株：S&P500 との比較を実施

⚠️ モデルの限界・注意事項

  • 本モデルは中長期バリュー選定に重みを置いた設計です。短期トレードには別途テクニカル分析をお勧めします
  • 流動性・市場影響コスト・スリッページは考慮していません
  • ファンダメンタルデータはyfinance経由で取得するため、最大1日の遅延が発生します
  • セクター相対評価・ベンチマーク（TOPIX/S&P500）比較は現時点で未実装です
  • 機関投資家向けの流動性・時価総額フィルターは未実装です
  • 本スコアはあくまで参考情報であり、特定銘柄の投資を推奨するものではありません

📄 今後の改善予定

  • 時価総額・流動性フィルター（機関投資家向け閾値）
  • バックテスト期間の延長（現在：直近6ヶ月 → 目標：3年）
  • ESG/ガバナンスフラグの統合
