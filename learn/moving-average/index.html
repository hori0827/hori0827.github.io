移動平均線とは？ゴールデンクロス・デッドクロスをわかりやすく解説

📋 この記事のポイント
  • 移動平均線は一定期間の終値の平均値を結んだ線。トレンドを視覚化する
  • ゴールデンクロス（短期線が長期線を上抜け）は上昇トレンドへの転換シグナル
  • デッドクロス（短期線が長期線を下抜け）は下降トレンドへの転換シグナル
  • 株価が移動平均線の上にあれば上昇トレンド、下にあれば下降トレンド

移動平均線とは何か

移動平均線（Moving Average）は、過去N日間の終値の平均値を毎日計算して折れ線グラフにしたものです。日々の細かい値動きのノイズを除いて、相場の大きな方向性（トレンド）を把握するために使います。

よく使われる移動平均線の種類

種類期間使いどころ
短期線5日・25日短期トレンド・売買タイミングの判断
中期線75日中期トレンドの方向確認
長期線200日長期の強気・弱気相場の判断

ゴールデンクロスとデッドクロス

ゴールデンクロス（買いシグナル）

短期移動平均線（例：25日線）が長期移動平均線（例：75日線）を下から上に突き抜けるときに発生します。相場が上昇トレンドに転換するサインとして多くの投資家が注目します。

デッドクロス（売りシグナル）

短期移動平均線が長期移動平均線を上から下に突き抜けるときに発生します。下降トレンドへの転換を示す売りシグナルです。

移動平均線を使った基本的な判断方法

  • 株価が移動平均線より上にある → 上昇トレンド（買いが有利）
  • 株価が移動平均線より下にある → 下降トレンド（様子見・売りが有利）
  • 移動平均線が右肩上がりで株価がその上 → 強い上昇トレンド

HG AnalyticsのAIは、各銘柄のトレンド判定に移動平均線を活用し、「上昇トレンド継続中」「調整局面」などのラベルを付与しています。
